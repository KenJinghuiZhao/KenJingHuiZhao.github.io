1.App introduction:

KenBrowser is a mobile browser that used in Android mobile devices.

Key features:
  • Tiny and clean, as small as only 4M in size.
  • The unique Floating Button to control webpage swipe, and auto-scroll.
  • Focuses on easy/quick moves on daily webpage browsing, with only one finger!
  • Bookmarks can be hidden.
  • Support self-defined javascript injecting, support TamperMonkey scripts.

2.App main UI：

KenBrowser has only one main UI, with title bar in top, and button bar in bottom. Some buttons support both click and long-click actions.


3.Floating button：

Floating button is an unique feature in KenBrowser, with which to control all kinds of webpage scroll and swipe.

Click:Click the floating button, will scroll webpage up.
Dbl-click: Will trigger webpage auto-scrolling. Can adjust scroll speed with popup buttons. Click floating button again to stop.
Swipe:Touch the button and swipe out of the button boundery, will trigger webpage swipe in 4 directions accordingly.
Lng-click: Long-click button till a small vibrate, then drag the button and drop anywhere you want!


4.Hide bookmarks

In the KenBrowser homepage, double-click the "Bookmark" title, will show the hidden bookmarks. (there is one hidden bookmark embedded with KB)
Enter the "Favorite list" dialog (system menu, "Favorite List"), double-click the "Favorite List" title, will also show the hidden bookmarks in the list. When showing hidden bookmarks, click the menu button and choose "Toggle hide/show" to change bookmark status.


5.Injecting javascript

KenBrowser is one of those few mobile browsers that support javascript injecting in webpage. It supports TamperMoney scripts as well.
In JS List dialog (System menu, "Setup"-"Injected Javascript"-"JS List"), can check the scripts that installed (a default js is installed as Zhihu website mobile enhancement)
Click the item menu, will show menus to Delte, Modify and Share the javascripts.

There are 2 ways to import javascript:
1. In your mobile's file management app, choose the js file, and share/send to KenBrowser.
2. In KenBrowser system menu, "Setup"-"Injected Javascript"-"Import JS", in the file pickup UI choose the js file you want, click OK.

Hint: the imported javascript file should complie with TamperMoney script standard, i.e. filename and file head comments.

In Greasy Fork, there are enormous amount of user scripts, that can extend KenBrowser functions unlimitely. You can check author's uploaded scripts, and your are welcome to upload your scripts as well:)


6.网页文本选择

因为技术性的原因，现时在网页选择文本的操作有一点点复杂。日后将会改进。
操作：长按网页空白位置，弹出菜单-"选择文本"，进入文本选择模式。然后再长按文本选择文本。

7.分享到KenBrowser

你可以在其它app分享内容到KenBrowser。现在KB支持以下格式的分享：
纯文本： 在其它app选择文字或单词，分享到KB，将直接打开该关键字的搜索结果。
html文件： 可以把一个html文件分享/发送给KenBrowser，将会打开该网页。
js文件： 参考【链接】。
剪贴板url： 在任意app复制url到剪贴板，然后进入KenBrowser，点击顶部标题栏的网址输入框，将提示问是否打开网址。


8.从KenBrowser分享

在网页任意位置长按，在弹出菜单选择“保存/分享”，可以把当前网页的文本，或html源码分享给其它程序。

在其它浏览器打开本页：KenBrowser专注于网页浏览，所以某些功能可能别的浏览器会更强大。你可以通过长按弹出菜单-“保存/分享”-“其它浏览器打开”， 随时在别的浏览器打开当前网页，以便利用其它浏览器的强大功能。


9.收藏夹书签的导出导入

KenBrowser支持收藏夹书签的导入和导出。

导出：在主页长按弹出菜单，选择“保存/分享”-“分享本页html..”，就可把收藏夹分享给其它文本编辑app或手机文件管理器。然后在导出的文件上做插入、删除和修改。

导入：把修改后的文件保存，然后分享/发送给KenBrowser，则KB会把文件里的书签融合到你现有的书签列表里。在主页刷新则可看到融合后的结果。

注意：修改书签html文件时，只能修改书签部分，html的其它部分不要随意改动，否则KenBrowser不能正确识别。
